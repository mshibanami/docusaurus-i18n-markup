# testsite

This is a test site for `docusaurus-i18n-markup`.

See the [main README](../README.md) for the details.

## How to Run

1. Build `docusaurus-i18n-markup` and `docusaurus-plugin-write-translations-extended` packages first. From the root directory of this repository, run:

   ```bash
   pnpm install
   ```

2. Start the site in the `testsite` directory:

   ```bash
   pnpm start
   ```
